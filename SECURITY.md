# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in LatticePHP, please report it responsibly.

**Email:** security@latticephp.dev

Please do NOT open a public GitHub issue for security vulnerabilities.

## Response Timeline

- **Acknowledgement:** Within 48 hours
- **Initial assessment:** Within 5 business days
- **Fix and advisory:** As soon as practical, coordinated with reporter

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | Yes       |
